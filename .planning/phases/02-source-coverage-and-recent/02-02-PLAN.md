<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| SQLite configuration to browser projection | A faulty serializer could broaden coverage, misclassify an update, or expose raw/server-only fields. |
| Browser selected location to filtered records | A UI mistake could show records from an unselected jurisdiction. |
| Publication date to feed presentation | Dates can be misused as policy/legal status claims. |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-02-06 | Information disclosure | browser model | medium | mitigate | Add only location IDs/labels/coverage; retain Phase 1 narrow record shape. |
| T-02-07 | Spoofing | feed status | medium | mitigate | Preserve source kind and citation fields; UI may not infer policy status from date. |
</threat_model>
