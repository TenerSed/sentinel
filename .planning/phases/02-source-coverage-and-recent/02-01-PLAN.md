<threat_model>
## Trust Boundaries

| Boundary | Description |
|---|---|
| Allowlist to network | A local maintainer command must not become an arbitrary URL fetcher or crawler. |
| Fetched bytes to import database | Remote material is untrusted and may be huge, malformed, duplicated, or lack citable evidence. |
| Import database to demo seed | Live material must not silently change the judge-facing corpus. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|---|---|---|---|---|---|
| T-02-01 | SSRF | manual importer | high | mitigate | Finite HTTPS allowlist, host validation before and after redirects, no URL input. |
| T-02-02 | Denial of service | remote response | medium | mitigate | Fixed response-size cap and independent source processing. |
| T-02-03 | Tampering | live material in feed | high | mitigate | Separate DB; evidence-less material is draft-only and no UI reads imports.db. |
| T-02-04 | Repudiation | imported material | medium | mitigate | Store configured/original URL, canonical URL, hash, retrieval timestamp, outcome, and diagnostic. |
</threat_model>
